# fmui-react

> 基于react的组件开发

## Build Setup

``` bash
# 安装依赖包
npm install

# webpack开发模式
npm run dev

# webpack打包生产环境
npm run build

# 开发模式，webpack+mock提供react需要的模拟接口数据，建议前端开发使用这个命令
npm run mockdev

```
